Commands what we used on Lesson 18

  • git init [name] - Create new local repository with name
  • git clone [url-adress] - Downloading repository with all changing history
  • git status - Show all files to be committed
  • git add [file] - Indexes the specified file for later commit
  • git commit -m "[description message]" - Commits indexed changes and saves them to version history
  • git branch - Show list of brsnches
  • git branch [branch name] - Create new branch
  • git switch -c [branch name] - Change branch
  • git push - Downloads all changes from the local branch to the remote repository
  • git pull - Downloads all history to lokal
